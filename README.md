# Timple.ai Enterprise Transformation Engine

This repository contains the complete pitch, strategy, and functional UI prototypes for the Timple.ai Technical Co-Founder pitch. 
The strategy focuses on pivoting Timple.ai into an **Apex Ecosystem** that leverages Open-Source local LLMs (Skyfall 31B), Graph-Database Memory, and the Civitai Creator Economy.

## Repository Structure

### 1. Strategy & Vision Documents
Located in `/pdf`, `/ppt`, and `/video-script`, these documents outline the complete market domination strategy for 2026.
*   **AI_Transformation_Blueprint.md:** The definitive threat landscape matrix comparing Candy.ai, Darlink, Secrets, and NeuralCompanion.
*   **Timple_AI_Strategy_Deck.md:** The slide deck ready for presentation.
*   **NotebookLM_Script.md:** The promotional video script.

### 2. Functional Prototypes (Next.js)
This repository contains two separate Next.js applications that bring the pitch to life.

#### A. The B2B Enterprise Website (`/website`)
A data-driven dashboard pitch site designed to prove the Lifetime Value (LTV) superiority of the Timple.ai ecosystem over competitors.
*   **To run locally:** `cd website && npm install && npm run dev`

#### B. The Consumer AI Prototype (`/prototype`)
A functional mock of the Next-Generation Timple.ai interface featuring:
*   Graph-Database Memory Vaults
*   Multi-Agent Spatial Invite System
*   NeuralCompanion Cloud Sync toggles
*   **To run locally:** `cd prototype && npm install && npm run dev`
